# Oxygen Sensor Diagnosis and Replacement

## Upstream vs downstream — know which one you're condemning

Upstream (Sensor 1) sensors control fuel trim; downstream (Sensor 2) sensors monitor the catalyst. Bank 1 is the bank containing cylinder 1 — verify bank identity for the specific engine before ordering, because V-engine bank conventions differ by make. A P0420 with lazy downstream switching does not automatically mean the downstream sensor is bad; compare both waveforms first.

## Diagnosis before replacement

1. Graph the upstream sensor at 2,500 RPM steady: a healthy conventional O2 switches rich/lean several times per second, swinging roughly 0.1-0.9V. Slow, lazy switching with no stored heater code is a worn sensor.
2. Air-fuel ratio (wideband) sensors don't switch — they hold a commanded value. Test by forcing rich (propane) and lean (vacuum leak) and watching the response follow.
3. Heater circuit codes (P0030-P0064 range) are circuit problems: test heater resistance (typically 3-10 ohms cold) and confirm power and ground before replacing the sensor.
4. A sensor contaminated by coolant (head gasket) or silicone (wrong RTV) will fail again — find and fix the contamination source.

## Replacement procedure

1. Work on a warm-but-not-hot exhaust. Penetrating oil on the sensor base, let it soak.
2. Use a proper O2 sensor socket or, better, a crowfoot flare wrench on stubborn sensors. If the sensor fights, run the engine to heat the bung, then crack it loose — heat expands the bung more than the sensor.
3. If the threads strip, chase the bung with an M18x1.5 thread chaser before installing the new sensor.
4. New sensor: most quality sensors ship with anti-seize pre-applied on the threads — do not add more, and never get anti-seize on the sensor tip.
5. Torque to spec — typically 30-37 lb-ft (40-50 Nm) for an M18 sensor. Overtightening distorts the body and shortens sensor life; hand-tight-plus-a-grunt is not a spec.
6. Route the harness exactly as factory, away from exhaust contact, and seat the connector until it clicks.

## After replacement

Clear codes and run the engine to operating temperature. Verify fuel trims settle within ±10% and, for a downstream replacement, confirm the catalyst monitor completes a drive cycle before releasing the vehicle — that's the proof the repair holds.
